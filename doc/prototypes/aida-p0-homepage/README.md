# Aida P0 首页 HTML 原型

这是基于 `doc/aida-requirement-thinking.md` 制作的方向评审原型，不是正式 PRD，也不是开发版本。

## 打开方式

直接用浏览器打开：

```text
doc/prototypes/aida-p0-homepage/index.html
```

## 原型验证重点

1. 四个角色首页是否明显不同；
2. 首页首屏是否能回答“我现在应该处理什么”；
3. 重点关注事项是否足够显眼；
4. 高关注 + 异常是否能优先进入首屏；
5. Token / Session 是否只作为辅助信息和工作证据；
6. 右侧抽屉是否足够承接首页点击，但没有扩成完整管理系统。

## P0 边界

- 提醒只包含站内曝光：卡片、Badge、排序、风险提示文案。
- 右侧抽屉动作都是 mock：关注 / 取消关注、更新状态、补充说明、标记已处理、绑定 Session、确认日报、Review 通过 / 打回。
- 不做完整需求管理、任务管理、日报管理、飞书集成、移动端、数字分身、自动派单、甘特图或 Token 大屏。
