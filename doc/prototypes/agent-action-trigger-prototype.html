后端解析链路

Action Resolver
1. 识别 action_key 从按钮、路由、定时任务或事件中得到同一个业务动作标识。
2. 装配上下文 读取当前用户、请求参数、日报上下文、任务和 session 数据。
3. 渲染 Start Prompt 用 Action 定义中的模板和参数映射生成 managed agent 入参。
4. 调用 Managed Agent 后端使用统一 token 调平台接口，保存运行记录和输出结果。
