## 1. 修复内容

本次修复解决 Report Agent 运行参数丢失问题。Aida 在创建平台 session 时会传 `Message`，平台在 `Message` 非空时不会渲染 Agent 的 `StartPromptTemplate`，导致真实模型首条消息里缺少 `run_id/report_type/period/target/mcp_url`，最终出现 `run_id does not exist` 或 `缺少 run_id 参数`。

修复后，Aida 在手动运行和定时运行两条路径都会先合并系统参数，再构造最终 `Message`，显式包含:

- `report_type`
- `period`
- `target`
- `run_id`
- `mcp_url`
- `credential slot` 使用说明
- 用户补充说明或默认报告生成说明
